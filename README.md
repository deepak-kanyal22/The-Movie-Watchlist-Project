# Movie Watchlist Project
## Project Description

The Movie Watchlist Project is a web application that allows users to search for movies and maintain a personalized watchlist. Users can explore movie details, add or remove movies from their list, and manage their viewing preferences.

This project demonstrates JavaScript concepts, API integration using fetch, and UI development, along with dynamic features like searching, filtering, and sorting.

## Objective

The objective of this project is to:

- Integrate public APIs using the fetch method
- Use JavaScript array higher-order functions (map, filter, sort)
- Build an interactive and visually appealing user interface
- Handle dynamic data and user interactions efficiently

## APIs Used
- Watchmode API – https://api.watchmode.com/
  
## Core Features
1. Live Search
```
Fetch movie data from the API dynamically
Display results as the user types or on search click
```
2. Movie Cards
```
Show movie poster, title, year, and rating
Each movie displayed as a card
```
3. Empty States
```
Display messages when:
No movies are found
Watchlist is empty
```
4. UI Design Ideas
```
Dark-themed UI inspired by streaming platforms
Clean and modern layout
Sticky sidebar for saved/watchlist movies
Responsive design
```

## Optional Feature

### Random Movie Night

Picks a random movie from the watchlist
